NPU task
========


Макросы контекста пакета
-------------------------
======== =========
Макрос   Значение
======== =========
-------- ---------
HEADER   Адрес памяти, хранящей заголовок пакета
-------- ---------
PHV      Адрес памяти, хранящей вектор полей разобранного заголовка
-------- ---------
PORTMASK Адрес памяти, хранящей маску выходных портов для пакета
======== =========
